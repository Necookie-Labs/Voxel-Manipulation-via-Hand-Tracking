# Contributing

Thanks for considering contributing to this project.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your changes.
3. Make your improvements or fixes.
4. Commit with clear and descriptive messages.
5. Open a Pull Request describing your changes.

## Guidelines
- Keep code clean, readable, and consistent with the project style.
- Make focused changes (avoid large unrelated commits).
- Test your changes before submitting.

By contributing, you agree to follow the project's Code of Conduct.
